Murder mystery

Problem Statement:

Mr. Ranjit is an intelligence officer. He was assigned a murder case and given all witnesses found at site of the murder. After analyzing all witnesses given to him, he found victim's personal diary in which officer found something scrawled in the corner of a page. Now, Mr. Ranjit believes that the scrawled text is more mysterious and can help in solving the case. So he decides to find every occurrence of all the permutations of the scrawled text in the entire personal diary.

Since this is a huge task, Mr. Ranjit wants to take his colleague's help. Consider yourself as his colleague who helps him to figure out if any permutation of the scrawled text exists in the given text string, so he can save time and analyze only those text strings where a valid permutation is present.

Input Format:

First line contains number of test cases T. Next, each test case contains two lines, first line contains the pattern and next line contains a text string. All characters in both the strings are in lowercase only [a-z].

Output Format:

For each test case print YES or NO depending on whether any permutation of the pattern exists in the text string.

Constraints:

1 ≤ T ≤ 100
1 ≤ |Pattern| ≤ 1000
1 ≤ |Text String| ≤ 100000

Sample Input:

3
hulk
mounthulk
teddy
come
coder
urredoc

Sample Output:

YES
NO
YES

Explanation:

hulk is the part of the scrawled text : mounthulk so it prints yes.
teddy is not the part of the scrawled text: come so it print no.
coder is the part of the scrawled text: urredoc so it print yes.
